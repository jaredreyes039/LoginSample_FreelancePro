## Login/Registration Sample for FreelancePro

### Tech Stack
- react v19.2.0
- tanstack/react-router v1.132.0
- tanstack/react-router-devtools v1.132.0
- react-hook-form v7.71.1
- tailwindcss v4.0.6
- axios v1.13.3

### Form Validation with Yup
Custom Yup ((jquense/yup)[https://github.com/jquense/yup])  resolver implemented to handle form validation for both registration and login.
